# MCQ Generator — IGCSE / A-Level Computer Science

Дообучение LLM для генерации тестовых вопросов с вариантами A-D.

Текущая модель — **QLoRA + Qwen2.5-7B-Instruct**: большая модель уже «знает»
Computer Science, дообучение только прививает формат экзамена, поэтому
вопросы фактически корректны. Старый вариант на FLAN-T5-large придумывал
неверные вопросы; он полностью удалён.

## Порядок запуска (QLoRA, актуальный)

```
# 1. Сбалансировать датасет (убирает перекос "ответ всегда B" и дубликаты)
python prepare_dataset.py

# 2. Быстрая проверка пайплайна (~2 минуты, модель 0.5B)
python train_qlora.py --smoke

# 3. Полное обучение (скачает ~15 ГБ базовой модели, затем несколько
#    часов на RTX 3060; создаст папку mcq_qlora с LoRA-адаптером)
python train_qlora.py

# 4. Генерировать вопросы
python generate_qlora.py
```


## Установка на компьютере с GPU (RTX 3060)

```
pip install transformers scikit-learn accelerate peft bitsandbytes
pip install torch --index-url https://download.pytorch.org/whl/cu126
```

Проверка, что GPU виден:

```
python -c "import torch; print(torch.cuda.is_available())"
```

Должно напечатать `True`. Если `False` — установлена CPU-версия torch,
переустановите второй командой выше.

## Файлы

| Файл | Назначение |
|---|---|
| `MCQ_clean1.txt` | исходный датасет, собранный вручную (~3400 вопросов) |
| `prepare_dataset.py` | чистка, удаление дублей, балансировка букв ответов |
| `MCQ_balanced.txt` | результат — обучающий датасет (создаётся скриптом) |
| `train_qlora.py` | QLoRA-обучение Qwen2.5-7B-Instruct (актуальное) |
| `generate_qlora.py` | интерактивная генерация вопросов (QLoRA-модель) |
| `mcq_qlora/` | сохранённый LoRA-адаптер (создаётся обучением) |

## Почему модель раньше всегда отвечала "B"

В исходном датасете правильный ответ был "B" в 41% вопросов, а "D" —
только в 5%. Модель выучила частоту букв вместо содержания.
`prepare_dataset.py` перемешивает варианты каждого вопроса так, что
правильные ответы распределены ровно по 25% на каждую букву, и пишет
каждый вопрос дважды с разными позициями правильного ответа.
